Eckhardt Lab Ansatzrechner

Dieses Tool hilft beim Zusammenstellen eines Reaktionsansatzes, berechnet Mengen/Volumina und sammelt Sicherheitsinformationen. Zuerst können unten Chemikalien über die Suche gefunden oder auch manuell angelegt werden. Im 2. Abschnitt „2. Ansatzrechner“ können die Einträge weiter editiert werden.

1. Chemikalie suchen (PubChem)

Suche nach Name, CAS, Summenformel oder SMILES in PubChem. Klick auf eine Ergebniszeile fügt das Reagenz dem Ansatz (Abschnitt 2) hinzu.

Bereit.
Name SMILES CAS Summenformel Molare Masse Dichte

2. Ansatzrechner

Lege die Ansatzgröße fest und bearbeite die Felder per Doppelklick. Eq bleibt immer editierbar. Die Berechnung von m und V erfolgt beim Verlassen der Felder für M bzw. Dichte. Alle Einträge sind durch Doppelklick editierbar. Per drag and drop können die Positionen im Ansatz getauscht werden. Beim Klicken des x am Ende einer Zeile wird die komplette Zeile gelöscht. Die Buttons am Ende sind hoffentlich selbsterklärend.

3. Reaktionsgleichung

Erzeugt aus den im Ansatz gewählten Edukten/Produkten sowie Reagenzien/Bedingungen eine Reaktionsgleichung als CDXML und SVG (ChemEq).

4. Sicherheitsinformationen für diesen Reaktionsansatz

Hier werden alle Reagenzien des Ansatzes ausgeführt. Die Tabelle unten soll wie gewohnt vor dem Durchführen der Reaktion selbstständig ausgefüllt und als SVG exportiert werden. H- und P-Sätze sollten primär aus GESTIS entnommen werden. Mit der Onlinesuche werden Infos aus PubChem zusätzlich aufgeführt (bitte prüfen).

GESTIS Import (Bookmarklet)

1) Öffne den GESTIS-Link (Spalte „Sicherheit“) per Einfachklick – der Tab muss vom Ansatzrechner aus geöffnet werden (ohne „noopener“), damit der Import zurücksenden kann.
2) Klicke im GESTIS-Tab das Bookmarklet „GESTIS → Ansatzrechner“.
3) Die H-/P-Sätze, Piktogramme, Signalwort und (wo erkannt) ein Eigenschafts-Auszug werden automatisch übernommen.
GESTIS → Ansatzrechner Tipp: Link als Lesezeichen speichern (Bookmarks-Bar) oder Code unten kopieren.
Falls dein Browser den window.opener blockiert (z. B. wenn du GESTIS nicht aus dem Ansatzrechner heraus geöffnet hast): Das Bookmarklet kopiert stattdessen JSON in die Zwischenablage. Füge es hier ein und klicke Import.
